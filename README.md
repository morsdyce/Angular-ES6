Angular-ES6 (WIP)
=================

Boilerplate for using angularjs with ES6 today.

Contains:
Angularjs
Angular UI Router
Foundation 5
jQuery
Gulp

Install
=======

``` git clone https://github.com/morsdyce/Angular-ES6.git && cd Angular-ES6 ```

``` npm install && bower install ```

Using
=====
Development:
``` gulp watch ```

build:
``` gulp build ```
